# CommandCube - Professional Remote PC Control

## What You Get

A complete remote control system for your PC with:
- **25 Premade Buttons** across 6 categories
- **Web Interface** (works in any browser)
- **Android APK** (control from phone)
- **Professional GUI Launcher** (easy management)
- **Production-Ready Server** (stable, crash-proof)

## Quick Install

### Option 1: Easy Install (Recommended)
1. Run: `python install.py`
2. Follow prompts
3. Open 'START.bat' to launch

### Option 2: Manual
1. Run: `python launcher.py`
2. Click "START SERVER"
3. Open browser: http://localhost:8765

## How to Use

### From PC (Web Browser)
1. Start the server via launcher
2. Open: http://localhost:8080
3. Click buttons to control PC

### From Phone (Android)
1. Transfer: CommandCube-Remote.apk
2. Install (Settings > Security > Unknown Sources)
3. Open app and enter PC IP
4. Control from phone
