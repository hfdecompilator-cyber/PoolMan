# CommandCube Server Pro - Enterprise Edition

Professional self-hosted PC control solution for remote script execution.

## Quick Start

### 1. Install
```
python install.py
```
- Automatically installs Python dependencies
- Sets up environment
- Creates desktop shortcut

### 2. Run
```
python server_pro.py
```
Or double-click "CommandCube Server Pro" on Desktop

### 3. Connect from APK
- Enter your PC IP (e.g., 192.168.1.100)
- Port: 5000
- Press Connect
- Run scripts remotely
